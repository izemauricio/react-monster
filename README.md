# react-web-monster
React web application to practice

## init
- npx create-react-app react-monster
- cd react-monster
- yarn start
  - creates the public folder from the workspace src
  - inject jsx `<app>` into dom `<div id=root>`

## deploy
- npm run build
  - creates the optimized build folder
  - babel and webpack

## advanced
- yarn eject

## architecture
- lifting state up
